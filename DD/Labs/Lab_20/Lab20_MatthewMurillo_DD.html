In this lab, its going to be a guessing game. There is a number that you, the player, have to guess. There will be a limit to how far you can guess. In this game, the number would be from 1 to 50. If the number you guess is lower than the answer, then the game would state a message saying to guess higher. If the number is higher than the answer, the game would say to guess lower. Guess the correct number and you win. Good luck and have fun!
